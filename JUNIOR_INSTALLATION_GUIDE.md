# 🚗 ParkSmart Pro: Junior Developer Installation Guide

Welcome to the **ParkSmart Pro** project! This guide will walk you through exactly how to set up the entire parking management system locally on your Windows machine from scratch. 

Follow these steps carefully, and you'll have both the Backend server and Frontend dashboard running perfectly in under 10 minutes.

---

## 🛠️ Prerequisites Before You Start

Make sure you have Python installed on your Windows machine:
1. Go to [python.org/downloads](https://www.python.org/downloads/)
2. Download the latest Python 3 installer.
3. **CRITICAL:** When running the installer, make sure to check the box that says **"Add Python to PATH"** before clicking Install.

---

## 🎯 STEP 1: Setting up the Backend

The Backend powers the database, API endpoints, Razorpay mock integrations, and the auto-reset features. It is built using Python, FastAPI, and SQLite.

1. Open a new **Command Prompt (CMD)** or **Terminal** window.
2. Navigate into your project's `backend` folder:
   ```cmd
   cd C:\path\to\your\Smart-Parking-System-main\backend
   ```
3. Create a clean Python Virtual Environment (this isolates our project packages):
   ```cmd
   python -m venv venv
   ```
4. Activate the virtual environment:
   ```cmd
   venv\Scripts\activate
   ```
   *(You should now see `(venv)` at the beginning of your terminal prompt)*
5. Install all required dependencies:
   ```cmd
   pip install fastapi uvicorn sqlalchemy pydantic pydantic-settings python-jose[cryptography] passlib bcrypt slowapi loguru razorpay email-validator httpx websockets
   ```
6. **Seed the Database:** We need to instantly create your Admin user and all the 48 parking slots (Zones A, B, and C). Run the seeder script:
   ```cmd
   python seed_parksmart.py
   ```
   *(It will automatically create `parksmart.db` and populate it!)*
7. **Start the Backend Server:**
   ```cmd
   uvicorn main:app --reload --host localhost --port 8000
   ```
   **Leave this terminal running!** Do not close it.

---

## 🌍 STEP 2: Setting up the Frontend

The Frontend is exactly what the users and admins see in the browser.

1. Open a **brand new** Command Prompt or Terminal window.
2. Navigate to the root directory where your frontend files are located (or where `login.html` and `dashboard.html` live):
   ```cmd
   cd C:\path\to\your\Smart-Parking-System-main\frontend
   ```
   *(If your HTML files are natively in the root folder, navigate there instead).*
3. Start the local Development Server for the frontend:
   ```cmd
   python -m http.server 5500
   ```
   **Leave this terminal running!** 

---

## 🚀 STEP 3: Launching the App

1. Open your web browser (Chrome, Edge, Brave, etc.).
2. Go to: **http://localhost:5500/login.html**
3. Log in using the Admin credentials generated by the seeder script:
   * **Email:** `admin@parksmart.test`
   * **Password:** `admin123`

You're in! You should see the sleek Dark Mode interface with dynamic Slot grid layout and statistics!

## 🧪 Testing the Advanced Features

As a junior debugging the app, test these functionalities to understand how the system clicks together:
1. **QR & Mock Payments:** Click any Green spot to book it. Create an end time strictly *after* the start time. When you click **Pay & Confirm**, you will see a Mock Checkout finalize instantly without external Razorpay windows crashing.
2. **Auto-Reset Testing:** Every 3 seconds, the browser will seamlessly grab live updates. If a booking's `end time` natively expires past your actual local time, the backend will quietly free the slot and auto-update your UI back to Green!
3. **Live Revenue Tracker:** Every time you mock-pay for a slot, look at the purple card at the top right—your dynamic Rupee (₹) revenue goes up immediately!
4. **Admin Privileges:** If you've been doing extensive testing, you can hit the red **Clear Slots (Admin)** button to totally wipe your parking bays clean and reset all stats across the board.
